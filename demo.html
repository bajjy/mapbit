🚀 Quick Start:

  1. Click "Launch App" to open the main application
  2. Click "📍 Load My Location" to set your location
  3. Click "🏠 Load Buildings" to fetch OSM data
  4. Click "✨ Process for RPG" to apply retro styling
  5. Use the sidebar controls to customize the appearance

🛠️ Development:

To start the development servers:

npm run install:all
npm run dev
